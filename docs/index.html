HTTP

Utilities for working with HTTP status codes, errors, and more

This package provides a series of useful Node.js modules meant for use in web servers. The following libraries are included:

  1. status - HTTP status code utilities
  2. errors - Custom HTTP errors for use in Koa, Express, etc.

Installation

Install using npm:

$ npm install --save @greylocklabs/http

Usage

Here's a basic web server using the Koa framework as an example:

import Koa from 'koa';
import Router from 'koa-router';

import { errors, status } from '@greylocklabs/http';

const app = new Koa();
const router = new Router();

router.get('/api', async (ctx, next) => {
    await next();

    ctx.status = status.OK; // 200
    ctx.message = status[200]; // 'OK'
    ctx.body = {
        api: 'running',
    };
});

router.get('/404', () => {
    throw new errors.NotFoundError('Nothing to see here!');
});

router.get('/500', () => {
    throw new errors.InternalServerError('Something broke!');
});

app.use(router.routes());

app.listen(3000);
